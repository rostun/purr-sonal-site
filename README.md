# purr-sonal-site 🌹

This is my personal website, built with Next.js.

It doubles as a small infrastructure experiment: the same app is deployed to both Vercel and Cloudflare to compare platform tradeoffs, edge behavior, and developer experience.

## Tech

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Goals

- Keep the site simple and intentional
- Explore differences between hosting platforms
- Avoid overengineering a personal website (harder than it sounds)

## Status

In progress — currently getting the basics in place before layering on content.
